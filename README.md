# CASA MORI

## Project Description
A portfolio page that takes you to my socials, contact, and projects.

## Intended User or Player
People who are looking to connect with an multi-disiplinary artist or are just curious about me.

## User / Player Goal
The user will be able to see my work and learn more about me.

## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.

## Design Choices
- Layout: Bento layout design, title, nav, bio, gallery, ethos, and resources
- Color: Greyscale
- Typography: Poppins font
- Visual hierarchy: The bio and gallery sections are the largest and are meant to grab users attention first
  Color: Greyscale
- Feedback states: Hover states on global navigation and Ethos section
